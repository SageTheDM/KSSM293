426 Software Development with Agile Methods

This module covers agile software development in release cycles. It is part of the educational requirements for various IT-related certifications.

Competence

Applying agile methods for software development in release cycles.

Objectives

  • Implement functionality using an agile method.
  • Develop software step by step with best practices.
  • Use existing design patterns and software components effectively.
  • Reflect on results and improve processes.
  • Use version control systems.
  • Write clean and understandable code.
