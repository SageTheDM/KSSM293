Lexikon

  • Wiki(pedia) – A collaborative online encyclopedia where users can edit and contribute.
  • CSS – Cascading Style Sheets, used to style HTML elements.
  • HTML – HyperText Markup Language, the standard for structuring web content.
  • User Generated Content – Content created by users rather than website owners.
  • Google Docs – An online document editor by Google.
  • PHP – A server-side scripting language for web development.
  • API – Application Programming Interface, allows communication between software.
  • XML – Extensible Markup Language, used for data structuring.
  • Mobile Endgeräte – Mobile devices like smartphones and tablets.
  • Web Services – Services that allow applications to communicate over the internet.
  • Document Object Model (DOM) – A programming interface for web documents.
  • JavaScript – A scripting language for interactive web applications.
  • Web – The World Wide Web, a system of interconnected web pages.
  • Communities – Online groups where users share interests.
  • Twitter – A social media platform for short messages (tweets).
  • SQL – Structured Query Language, used for database management.
  • Datenbank – The German word for "database."
  • JSON – JavaScript Object Notation, a lightweight data format.
  • Podcast – Digital audio content distributed online.
  • Repository – A storage location for code, often in version control systems.

  • jQuery – A JavaScript library that simplifies DOM manipulation and event handling.
  • Cookies – Small pieces of data stored on a user's device by websites.
  • Bootstrap – A framework for responsive web design.

  • AJAX – Asynchronous JavaScript and XML, used to update parts of a web page without reloading.
  • XING – A professional networking platform, similar to LinkedIn, popular in German-speaking countries.
  • Flickr – A photo-sharing website.
  • Semantisches Web – "Semantic Web," a concept where web data is structured to be more understandable for machines.
